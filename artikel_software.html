Jika di artikel sebelumnya kita membahas **Hardware** sebagai tubuh fisik komputer, sekarang kita akan membahas **Software**. *Software* adalah "jiwa" atau "pikiran" yang memberi tahu *hardware* apa yang harus dilakukan.

Apa Itu Software (Perangkat Lunak)?

Pengertian: Software (Perangkat Lunak) adalah kumpulan data elektronik, instruksi, atau program yang disimpan dan dijalankan oleh komputer untuk melakukan tugas tertentu. *Software* tidak memiliki bentuk fisik dan tidak bisa disentuh.

Analogi Restoran: Jika *Hardware* adalah dapur, kompor, dan koki, maka *Software* adalah **buku resep** dan **manajer restoran**. Resep memberi tahu koki (CPU) cara mengolah bahan (data). Manajer (Sistem Operasi) mengatur siapa melakukan apa dan kapan.

1. Perangkat Lunak Sistem (System Software)

Ini adalah "Manajer Restoran". Tugasnya adalah mengelola semua sumber daya *hardware* dan *software* di komputer. Tanpa *software* sistem, komputer tidak bisa menyala.

A. Sistem Operasi (Operating System / OS)

Pengertian: *Software* paling penting yang bertindak sebagai "jembatan" antara kamu, aplikasi, dan *hardware*.

Fungsi: Mengelola memori (RAM), mengatur file (di SSD/HDD), menangani input (dari Keyboard), dan menampilkan output (ke Monitor). Dialah "bos" dari semua *software*.

Contoh: Microsoft Windows, Android, iOS (iPhone), MacOS (Apple), Linux.

B. Perangkat Lunak Utilitas (Utility)

Pengertian: *Software* yang bertugas sebagai "tim kebersihan dan keamanan" komputer.

Fungsi: Membantu merawat komputer. Tugasnya spesifik seperti melindungi dari virus, membersihkan file sampah, atau mengatur data di *hard drive*.

Contoh: Antivirus (Avast, Norton), File Manager (Windows Explorer), Disk Defragmenter.

2. Perangkat Lunak Aplikasi (Application Software)

Ini adalah "Buku Resep" untuk tugas-tugas spesifik. Ini adalah program atau "Aplikasi" (Apps) yang kamu gunakan sehari-hari untuk menyelesaikan pekerjaan, belajar, atau hiburan.

A. Pengolah Kata (Word Processor)

Fungsi: Untuk membuat dan mengedit dokumen teks, seperti mengetik surat, laporan, atau cerpen.

Contoh: Microsoft Word, Google Docs, WPS Writer.

B. Peramban Web (Web Browser)

Fungsi: Untuk menjelajahi dan menampilkan halaman web di internet. Kamu menggunakan ini untuk membuka Google, YouTube, atau Facebook.

Contoh: Google Chrome, Mozilla Firefox, Microsoft Edge, Safari.

C. Pengolah Presentasi (Presentation)

Fungsi: Untuk membuat *slide-slide* yang ditampilkan saat presentasi di depan kelas.

Contoh: Microsoft PowerPoint, Google Slides, Canva.

D. Game (Permainan)

Fungsi: *Software* yang dibuat khusus untuk hiburan.

Contoh: Minecraft, Mobile Legends, Roblox, dan game-game lain di HP atau komputermu.
